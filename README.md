# 📌 Project Overview

**My Portfolio** is a modern and responsive personal portfolio website developed to professionally showcase:

* 👨‍💻 Skills
* 🚀 Projects
* 📄 Resume / CV
* 📬 Contact Information
* 💼 Professional Experience

This portfolio was created using **HTML5, CSS3, JavaScript, and Google Apps Script** with a clean UI design and mobile-friendly layout.

The website represents my journey as a **Computer Engineering student and aspiring software developer**.
